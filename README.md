# 🧬 IntelliPRRSV2

A scalable genomic intelligence platform for PRRSV mutation analysis, lineage tracking, and vaccine escape prediction.

## Features
- Mutation hotspot detection
- ORF-level mutation analysis
- Phylogenetic tree construction
- Lineage classification
- Vaccine escape modeling
- miRNA interaction analysis
- Geographic mapping
- Streamlit dashboard

## Run Dashboard
streamlit run dashboard/app.py
